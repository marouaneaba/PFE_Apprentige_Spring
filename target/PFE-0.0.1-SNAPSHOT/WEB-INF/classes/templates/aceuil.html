Appretissage d'algorithme

vous étes dans une application pédagogique d’initiation à l’algorithmique A l’aide d’un langage algorithmique en français et des interface graphique simple, cette application permet de concevoir et de tester facilement les algorithmes que l’on peut rencontrer dans l’enseignement des mathématiques au secondaire. Le code source des algorithmes se construit pas à pas à l’aide de commandes prêtes à l’emploi (lire, afficher, affecter, instruction si…alors, boucles pour…de…à et tant que…) : cela permet de se concentrer principalement sur la logique algorithmique plutôt que sur l’apprentissage d’une syntaxe complexe. Une fois l’algorithme mis au point, vous pouvez transmettre votre solution à un enseignant .
